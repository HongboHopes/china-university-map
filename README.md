# 中国大学探索地图 🎓

覆盖 **1166 所**本科院校的高考志愿填报辅助工具，基于高德地图和教育部公开数据。

## 快速使用

1. 下载本项目
2. 双击 `index.html`（需联网加载高德地图瓦片）
3. 推荐使用 Chrome / Edge 浏览器

## 功能

- 🗺️ 中国地图标注所有本科院校（高德地图底图 + 省界线）
- 🔍 搜索校名 / 按省份、办学层次、学校类型筛选
- 📍 点击省份聚焦放大，选中省份高亮，其余省份虚化
- 📋 点击学校查看信息卡片：
  - 基本信息（省份/城市/类型/层次/建校年份/办学性质）
  - ⭐ 双一流建设学科 / 王牌专业
  - 📊 第四轮学科评估（A+ / A / A-）
  - 🏠 食宿校园 + 气候 + 男女比例（仅供参考）
  - 🌐 招生官网 + 📖 百度百科一键直达
- 🏫 多校区大学独立标注（如中山大学珠海/深圳校区）
- 🖱️ 悬停显示校名，滚轮缩放至街道级

## 数据来源

| 信息 | 来源 | 可靠性 |
|------|------|:----:|
| 坐标 | 高德地图地理编码 API | ★★★★★ |
| 双一流学科 | 教育部公布名单 | ★★★★★ |
| 学科评估 | 教育部第四轮评估 | ★★★★★ |
| 建校年份 | 公开校史记录 | ★★★★ |
| 招生官网 | 高德 POI + 域名匹配 | ★★★ |
| 食宿/气候/男女比例 | 基于地区/类型推断 | ★★ |
